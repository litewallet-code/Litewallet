## Update policy

LiteWallet follows these update practices:

- **Security updates** are released as soon as issues are identified and fixed
- **Feature releases** roll out platform-by-platform (desktop typically first, then mobile app store review cycle)
- **Cross-platform compatibility** is maintained — wallets created on any version can be restored on any newer version
- **Backward compatibility** for recovery phrases is guaranteed — a phrase created years ago still works today

## How to update

### Desktop (Windows, macOS, Linux)
Download the latest installer from [litewallet.dev/download](https://litewallet.dev/download) and run it. Your wallet data persists through updates.

### iOS
Updates arrive via the App Store. Enable automatic updates in iOS settings for the fastest rollout.

### Android
Updates arrive via the Play Store. Enable automatic updates in Play Store settings.

## Verifying downloads

For desktop installers, verify the download checksum against the values published on [litewallet.dev/download](https://litewallet.dev/download). This ensures the installer hasn't been tampered with in transit.
